# Changelog

## Current Alpha

### v6.3.0-alpha.1 (2026-04-10)
- Added a dedicated Profiles tab with saved profile management, import/export actions, and quick presets.
- Added preset flows for Adventure, Spooky, Minimal, and DonnieDice-curated profile setups.
- Expanded the options panel into a 3-row alphabetical tab layout with new placeholder modules for Combat, Collectibles, Loot, and Prey.
- Added a dedicated Debug module and tab for scoped troubleshooting.
- Improved Delve life-credit handling, nested dropdown behavior, popup safety, and addon metadata fallbacks.

See `docs/CHANGES.md` and `docs/changelogs/6.3.0-alpha.1.md` for the fuller release notes.
