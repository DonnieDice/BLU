## v7.0.0

### Database — RGX:NewDatabase Migration
- Switched from `RGX:OpenDB` to `RGX:NewDatabase` (RGX-Framework v1.9.0)
- `BLU.db` is now the NewDatabase proxy — `BLU.db.enabled` reads/writes directly via `__index`/`__newindex` metamethods
- Removed `Database._handle` — all profile CRUD delegates to the proxy
- Fixed profiles UI to not replace `BLU.db` proxy with raw table reference
- Kept backward-compat shims: `BLU.GetDB`, `BLU.SetDB`, `BLU.InitializeDatabase`

### Combat — Load Screen Safety
- Deferred legacy combat event registration with `C_Timer.After(0.5)` delay
- `InCombatLockdown` guard with `PLAYER_REGEN_ENABLED` retry fallback
- Fixes `ADDON_ACTION_FORBIDDEN` taint when loading into battleground/arena/PvP zones

### UI
- Volume slider overhaul with row layout fixes
- Housing panel enhancements
- Combat UI and profile management staging

### Platform
- TOC bump to 120005 (The War Within + Nightfall)
- Enhanced release workflow with Discord notifications
- README and docs polish
