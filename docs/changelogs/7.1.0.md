## v7.1.0

### Framework Migration - Deduplication
- Removed local `combat_protection.lua` (344 lines) - replaced by `RGX:QueueForCombat()`
- Removed local `dropdown.lua` (252 lines) - replaced by `RGX:GetDropdowns()` (876 lines, richer API)
- Dropdown call sites migrated: sound_panel, combat, profiles, general, design
- `ForceWidth` options remapped: `compactRightControl` -> `compactRight`, `deleteKey`/`previewKey` -> `inlineKeys`
- `StyleLastAddedButton` replaced by RGX's declarative `onButtonCreated` pattern
- `ResetLevel` replaced by `Drops:HideInlineButtons(level, key)`

### Platform
- Release workflow now triggers on tags only (no double CurseForge uploads from branch pushes)
- Fixed changelog encoding for CurseForge display
