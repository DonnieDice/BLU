# v6.1.3 Changelog

## Bug Fixes
- Fixed BLU default sounds fallback playing non-existent base `.ogg` files — all defaults now registered with `_med.ogg` suffix so both primary variant and fallback paths resolve to files that exist on disk

## New Features
- **Sound channel selection** for soundpack/game sounds: per-event dropdown (Master / SFX / Music / Ambient) appears in each sound tab when a non-BLU sound is selected; defaults to SFX
- BLU internal/default sounds now always play on the `Master` channel (volume controlled by Low/Med/High file variants as before)
- **Low / Med / High labels** added beneath the volume slider for clarity

## UI Improvements
- Volume slider, Test button, and channel dropdown now properly vertically aligned
- Housing tab spacing now matches all other tabs (consistent 10px inset)
- Header: Discord link moved below RGX Mods branding on the right side
- Header: Discord invite updated to `discord.gg/N7kdKAHVVF`
- Header: RGX color updated to burgundy (`#8b4b5c`)
