## v8.0.0

Stable BLU v8 release for WoW Retail 12.0.7.

### Framework Integration

- BLU now requires RGX-Framework and delegates events, timers, hooks, slash commands, database/profiles, dropdowns, and shared runtime helpers to RGX.
- `BLU.db` stays as the RGX database proxy and is no longer overwritten during profile switches.
- Combat-sensitive registration and playback paths route through safer framework/runtime helpers.

### Combat and Sounds

- Combat options are scoped to launch-ready triggers: Combat Start, Combat End, and Combat Music.
- User custom sound discovery now resolves short names, fuzzy names, BLU media paths, and generated/custom manifests through the same resolver.
- User Custom Sounds now display in the Sounds tab table from the same registry source used by the sound pack count.
- Combat music playback uses the selected custom sound route instead of forcing the wrong sound channel.

### Fixes

- Guarded quest debug handlers against nil quest IDs.
- Fixed nested dropdown preview button state so Play can become Stop during preview playback.
- Removed dead database/import/export shims that conflicted with the RGX proxy model.
