# v6.5.0 Changelog

## Combat Page
- **Combat now has a real options page**: The Combat tab now uses a full panel instead of the old placeholder/prototype flow.
- **Module toggle matches the rest of the addon**: Combat now has the same header-level enable/disable switch pattern used by the other sound pages.

## Combat Trigger Layout
- **Compact 2-column trigger cards are in place**: The page now uses the new card layout with room for 8 trigger options per page.
- **Paging is now part of the real panel**: Combat triggers can now be split across pages without falling back to the old mock overlap behavior.
- **Core combat cue rows are visible**: `Combat Start Sound`, `Combat End Sound`, and `Combat Music Track` now exist as real placeholder settings in the UI.

## Sound Selection UI
- **Combat rows now use the nested BLU sound menu structure**: The Combat dropdowns follow the same grouped hierarchy style as the event sound panels.
- **New row styling is preserved**: The updated compact dropdown, volume, and `Test` button layout stays in place while using the older dropdown formatting.

## UI Cleanup
- **Removed extra Combat explainer sections**: The `Combat Settings` and `Core Combat Cues` header blocks were removed so the actual trigger rows fit on the page cleanly.
