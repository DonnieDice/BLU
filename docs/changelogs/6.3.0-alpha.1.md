# v6.3.0-alpha.1 Changelog

## Updates
- Added a dedicated Profiles tab with saved profile management, import/export actions, rename/delete/reset flows, and copy-from-character support.
- Added preset support for DonnieDice, Spooky, Adventure, and Minimal setup flows so a profile can be configured quickly.
- Expanded the options panel into a 3-row alphabetical tab layout with dedicated placeholder modules for Combat, Collectibles, Loot, and Prey.
- Added a dedicated Debug module and tab with scoped troubleshooting controls.
- Improved Delve life-credit handling so life gain/loss sounds trigger more reliably.
- Tightened nested dropdown behavior so inline arrows, preview buttons, and delete buttons sit more naturally within submenu rows.
- Hardened popup editbox handling and addon metadata fallback logic to avoid nil-index and modern API lookup failures.
