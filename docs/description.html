# BLU | Better Level-Up! BLU is a Retail World of Warcraft addon that replaces default progression sounds with curated audio from many other games. Current version: `v6.3.0-alpha.1` *** ## Overview BLU lets you swap in custom sounds for major progression moments across World of Warcraft while keeping control over event categories, sound sources, and profile setups. This alpha focuses on a cleaner options experience, stronger profile management, preset-driven setup flows, and a dedicated Debug tab for scoped troubleshooting. *** ## Features - Achievement sounds - Battle Pet sounds - Delve sounds, including companion progression and lives events - Honor sounds - Housing sounds - Level-Up sounds - Quest sounds - Renown sounds - Reputation sounds - Trading Post sounds - Profile management with saved profiles and presets - Debug tab with scoped debug filtering - User custom sound support - External sound pack discovery and registration API support *** ## Highlights In v6.3.0-alpha.1 - Added a dedicated Profiles tab with saved profile management, import/export support, and presets - Added preset flows for quick setup swaps without hand-tuning every event - Expanded the options panel to a 3-row alphabetical tab layout with future-facing module placeholders - Added a dedicated Debug tab backed by its own module and scoped debug controls - Improved Delve life-credit handling and tightened nested dropdown behavior - Hardened popup and addon metadata safety for modern WoW API behavior *** ## Slash Commands ```text /blu /blu help /blu debug /blu status /blu enable /blu disable /blu refresh /blu rescan /blu addcustom myfile /blu removecustom path ``` *** ## Sound Pack Support BLU supports: - built-in BLU sound packs - external addon packs discovered automatically - direct registration through `BLU:RegisterExternalSoundPack(...)` - user custom `.ogg`, `.mp3`, and `.wav` files *** ## Compatibility - WoW Retail only - Interface version: `120001` For Classic support, use BLU Classic. *** ## Roadmap Planned follow-up areas include: - deeper debug tooling built on the new Debug tab - more compact and intuitive Profiles tab refinements - additional named presets and setup flows - continued trigger validation and event coverage cleanup *** ## Support - Discord: https://discord.gg/N7kdKAHVVF - GitHub: https://github.com/donniedice/BLU -
