# 🎮 RGX | Better Level-Up! ![BLU Logo](https://media.forgecdn.net/attachments/1260/966/638038186089895874-png.png) ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Better Level-Up!** changes up the repetitive sounds in World of Warcraft, giving you the option to hear sounds from other games when you level-up, complete Trade Post activities, gain reputation, or level-up your Delve Companion. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) **The Kiwi Says:** "Bwwiiiee." *** ## ✨ Features 🏆 Achievements - Play sounds from other games when you earn Achievements! 🐾 Battle-Pets - Play sounds from other games when you level-up Battle-Pets! ⛏️ Delve Companion Level-Up - Play sounds from other games when your Delve Companion levels-up! ⚔️ Honor Ranks - Play sounds from other games when you gain Honor Ranks! 📈 Level-Up - Play sounds from other games when you Level-Up! 🌟 Renown Reputation Ranks - Play sounds from other games when you gain Renown reputation ranks! 👥 Reputation Ranks - Play sounds from other games when you gain Reputation ranks! 📋 Quests - Play sounds from other games when you Accept or Turn-in Quests! 🏪 Trading Post Activities - Play sounds from other games when you complete Trading Post activities! 🎵 Soundpack Support - BLU auto-discovers compatible sounds from other loaded addons at startup — no external libraries required. Addon devs can also register packs directly using BLU:RegisterExternalSoundPack(). If LibSharedMedia-3.0 is installed, BLU hooks into it automatically for even broader coverage. 🗂️ User Custom Sounds - Drop your own .ogg, .mp3, or .wav files into Interface\AddOns\ or Interface\AddOns\sounds\ using names like custom01.ogg through custom24.ogg. BLU auto-picks them up with no Lua editing required. **Note:** All sounds feature independent volume adjustment capability. *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/blu # Opens the options panel
/blu debug # Toggles debug mode
/blu welcome # Toggles the welcome message on login
/blu help # Displays help information
/blu refresh # Rebuilds external and user custom sound pack registry
/blu rescan # Rescans for newly registered media (no reload needed)
*** ## 🎵 Soundpack Support & Developer API BLU v6.2.0 includes first-class soundpack support. BLU scans loaded addons for compatible audio at startup — no external libraries needed. If LibSharedMedia-3.0 is present it hooks in automatically. Addon developers can also register packs directly:
-- Register a soundpack directly with BLU from your addon
BLU:RegisterExternalSoundPack("My Pack Name", {
  { name = "Level Up", path = "Interface\\AddOns\\MyAddon\\sounds\\levelup.ogg" },
  { name = "Quest Done", path = "Interface\\AddOns\\MyAddon\\sounds\\quest.ogg" },
})
Use /blu refresh or /blu rescan to reload external packs live without restarting the game. *** ## 🗂️ User Custom Sounds Want to use your own audio files? BLU now supports drop-in custom sound slots with no Lua editing required:
Interface\AddOns\custom01.ogg
Interface\AddOns\sounds\custom02.mp3
Interface\AddOns\custom03.wav
Drop your .ogg, .mp3, or .wav files into Interface\AddOns\ or Interface\AddOns\sounds\ using custom01 through custom24 slot names, then use /blu refresh or /reload. They appear automatically in the User Custom Sounds pack in every module's sound dropdown. *** ## 🎵 Available Game Sounds
